Guia de Instalação - Triagem ODQ
=================================

1. Pré-requisitos
-----------------
- Instale o Python 3.10 ou superior: https://www.python.org/downloads/
- (Opcional) Instale o Git: https://git-scm.com/downloads

2. Instalação do Sistema
------------------------
- Copie a pasta do projeto para o computador desejado (exemplo: C:\Users\SeuUsuario\Downloads\Triagem)
- Abra o PowerShell na pasta do projeto
- Crie e ative o ambiente virtual:
	python -m venv .venv
	.\.venv\Scripts\Activate.ps1
- Instale as dependências:
	pip install -r requirements.txt

3. Configuração
---------------
- Preencha os arquivos de configuração necessários, como parameters.json e confidential_client_secret_sample.py, conforme instruções do projeto.

4. Executando o Sistema
-----------------------
- Para rodar o sistema, execute o arquivo iniciar_sistema.bat (duplo clique ou via PowerShell):
	iniciar_sistema.bat

5. Criando Atalho na Área de Trabalho
-------------------------------------
- Clique com o botão direito no arquivo iniciar_sistema.bat
- Selecione "Enviar para > Área de trabalho (criar atalho)"
- Renomeie o atalho se desejar

6. Suporte
----------
- Em caso de dúvidas, consulte o responsável pelo sistema ou o arquivo README_EXECUCAO.md
